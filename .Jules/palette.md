## 2025-05-14 - Localized Loading Feedback
**Learning:** Prioritizing localized loading feedback (e.g., spinning the refresh icon and disabling the button) over global page-blocking overlays improves perceived performance and keeps the user in their current context. For monolithic HTML files, using Python scripts with raw byte sequences for special characters (like '↺') is the most reliable way to perform multi-line or complex replacements.
**Action:** Always check for icon-only buttons that trigger async actions and apply localized feedback where possible.
