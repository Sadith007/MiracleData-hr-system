## 2025-05-14 - Accessibility Baseline for Monolithic Web Apps
**Learning:** In a single-file monolithic architecture, basic accessibility features like proper label associations and ARIA labels for icon buttons are often overlooked during rapid development, significantly impacting screen reader users and keyboard navigation.
**Action:** Always verify that every interactive element has either a text label or a descriptive aria-label, and ensure all form inputs are explicitly linked to labels via the 'for' attribute.
